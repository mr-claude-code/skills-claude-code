# Tópicos de Tecnologias do Curso

Este arquivo contém informações sobre as principais tecnologias abordadas nos cursos.

## Claude Code

**O que é:** CLI oficial da Anthropic para interagir com Claude, permitindo desenvolvimento assistido por IA diretamente no terminal.

**Conceitos principais:**
- Comandos CLI e workflows
- Integração com IDEs
- Automação de tarefas de desenvolvimento
- Skills customizadas
- MCP (Model Context Protocol) servers
- Hooks e configurações

**Aplicações práticas:**
- Desenvolvimento de aplicações
- Automação de código
- Criação de skills
- Integração com ferramentas

## Vibe Coding

**O que é:** Metodologia de programação que combina intuição, experimentação rápida e feedback iterativo.

**Conceitos principais:**
- Prototipagem rápida
- Desenvolvimento iterativo
- Experimentação com código
- Feedback loops curtos
- Design emergente

**Aplicações práticas:**
- Exploração de soluções
- MVPs e protótipos
- Aprendizado experimental
- Criatividade em código

## n8n

**O que é:** Plataforma de automação de workflows com interface visual (low-code).

**Conceitos principais:**
- Workflows visuais
- Nodes e conexões
- Automação de processos
- Integrações com APIs
- Triggers e schedules
- Execuções e logs

**Aplicações práticas:**
- Automação de tarefas
- Integração de sistemas
- ETL de dados
- Notificações automatizadas
- Web scraping
