# Estruturas Padrão de Conteúdo

## Estrutura de Cronograma Completo

Um cronograma detalhado deve incluir:

1. **Título e Visão Geral**
   - Nome do curso/módulo
   - Objetivos de aprendizagem
   - Público-alvo
   - Pré-requisitos
   - Carga horária total

2. **Módulos e Aulas**
   Para cada módulo:
   - Nome do módulo
   - Objetivos específicos
   - Duração estimada
   - Lista de aulas com:
     - Título da aula
     - Tópicos abordados
     - Duração
     - Tipo (teórica, prática, hands-on)

3. **Recursos e Materiais**
   - Exercícios práticos
   - Projetos
   - Leituras complementares
   - Ferramentas necessárias

4. **Avaliação**
   - Critérios de avaliação
   - Projetos finais
   - Exercícios de fixação

## Estrutura de Resumo de Tecnologia

Um resumo detalhado deve incluir:

1. **Introdução**
   - O que é a tecnologia
   - Para que serve
   - Quando usar

2. **Conceitos Fundamentais**
   - Principais conceitos
   - Terminologia essencial
   - Arquitetura/funcionamento

3. **Funcionalidades Principais**
   - Features mais importantes
   - Casos de uso
   - Exemplos práticos

4. **Instalação e Setup**
   - Pré-requisitos
   - Passos de instalação
   - Configuração inicial

5. **Exemplos de Código**
   - Exemplos básicos
   - Exemplos intermediários
   - Boas práticas

6. **Recursos Adicionais**
   - Documentação oficial
   - Tutoriais recomendados
   - Comunidade

## Estrutura de Tema/Ideia de Aula

Para gerar ideias de aulas:

1. **Título e Gancho**
   - Título chamativo
   - Por que este tema é relevante

2. **Objetivos de Aprendizagem**
   - O que o aluno vai aprender
   - Habilidades desenvolvidas

3. **Estrutura Proposta**
   - Introdução (5-10 min)
   - Conteúdo principal (30-40 min)
   - Prática hands-on (20-30 min)
   - Conclusão e próximos passos (5-10 min)

4. **Recursos Necessários**
   - Ferramentas
   - Materiais de apoio
   - Preparação prévia

5. **Exercícios Práticos**
   - Atividade guiada
   - Desafio independente
   - Projeto mini

## Estrutura de Infográfico

Prompts para infográficos educacionais devem incluir:

1. **Estilo Visual**
   - "Educational infographic"
   - "Modern, clean design"
   - "Professional but friendly"

2. **Conteúdo Central**
   - Título claro e grande
   - 3-5 pontos principais
   - Ícones ou ilustrações

3. **Paleta de Cores**
   - Cores corporativas ou temáticas
   - Alto contraste para legibilidade
   - Profissional

4. **Layout**
   - Hierarquia visual clara
   - Fluxo lógico de leitura
   - Espaçamento adequado

Exemplo de prompt:
"Create an educational infographic about [TOPIC] with a modern, clean design. Include [KEY POINTS]. Use a professional color scheme with blues and oranges. Layout should be vertical/horizontal with clear visual hierarchy. Icons and illustrations to represent concepts. Suitable for teaching and presentations."
